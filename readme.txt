Easly Team Profile Gallery

== Description ==

This plugin allows users to add their team profile in wordpress theme and manage their infomation. use Thumbnail picture, Destination, Description, tags, categories.

Shortcode:
[mw_team_gallery]

How it works:

1. After activation of MW Team Gallery you can see Team Member Tab in admin panel.
2. go to options menu of Team Member Tab.
3. copy Shortcode [mw_team_gallery].
4. paste where you want to show your team member galley.


Features:

2.your can add new Member.

3. your can edit member details.

4. Add/Edit/Remove categories.

5. Add/Edit/Remove tags



== Installation ==

Use WordPress' Add New Plugin feature, searching "MW Team Gallery", or download the archive and:

1. Unzip the archive on your computer  
2. Upload `MW-Team-Gallery` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Dashboard -> Team Member Tab and use it.
